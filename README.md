# 图书借阅系统 bookledingsystem

## 简介 Abstract

这是一个简单的基于Django的图书借阅系统，实现了基本的图书借阅管理功能。

项目用于结课作业，仅供学习交流使用。

包括：

1. 用户注册、登录、注销、修改密码、修改邮箱
2. 用户借书、还书、查看借阅记录

## 项目依赖 Dependencies

### 架构 Architecture

1. Django 5.0.6
2. Bootstrap 4.6.2
3. jQuery 3.6.0
4. MySQL 8.0.37
5. Python 3.12.3
